# User Manual

## Getting Started

1. Log in with your credentials.
2. Use the left navigation to access modules.

## Portal (Requester)

- Create an Incident
- Create a Service Request (Catalog)
- Search Knowledge
- Track your own tickets and request status

## Incidents

- List incidents and drill into details
- Add comments and track progress
- For requesters: only your own incidents are visible

## Requests

- Ticket Requests: service request tickets and their lifecycle
- Catalog Requests: submission record with approval/fulfillment status (agent view)

## Knowledge

- Search published articles
- Provide feedback (if enabled)

## Gamification

- View leaderboard, badges, achievements
- Redeem rewards (if your balance is sufficient)

## Agent Modules (Agent/Admin)

Agent users can access:

- Problems (Problem tickets + Known Errors)
- Changes (Change tickets + CAB calendar view)
- CMDB (services/relationships and impact)
- IT Assets (asset register)
- AI Agents (triage suggestions, impact)

## Tips

- Use Dashboard drill-through cards to jump directly to filtered work queues.
- Use “Create” quick actions where available to start a new record in one click.
